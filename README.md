# lillyGo-TTGO-T5-Arduino
Feature test for lillyGo TTGO T5 board on arduino

## Requires
* GxEPD library https://github.com/ZinggJM/GxEPD.git
* Adafruit Gfx https://github.com/adafruit/Adafruit-GFX-Library.git

### Pin
37 - Button 2
38 - Button 1
39 - Button 3
